# 6.1 화면 ID 체계

### 네이밍 규칙

```
SCR-{플랫폼}-{영역}-{순번}

플랫폼: W (Web), M (Mobile), A (Admin)
영역: AUTH, DASH, NOTE, CARD, GRAPH, SEARCH, BILLING, SETTINGS, ADMIN
순번: 3자리 숫자 (001~)
```

### 예시

| ID | 설명 |
|----|------|
| SCR-W-AUTH-001 | Web 로그인 화면 |
| SCR-M-NOTE-003 | Mobile 노트 편집 화면 |
| SCR-A-ADMIN-001 | Admin 대시보드 |
