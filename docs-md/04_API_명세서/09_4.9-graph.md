# 4.9 Graph 도메인

### GET /graph/data

그래프 시각화 데이터

```json
{
  "data": {
    "nodes": [
      { "id": "uuid", "title": "노트1", "linkCount": 5, "pageRank": 0.85 }
    ],
    "edges": [
      { "source": "uuid1", "target": "uuid2", "type": "wikilink" }
    ]
  }
}
```

### GET /graph/neighbors/{noteId}

특정 노트의 N-hop 이웃 조회

### GET /graph/clusters

자동 감지된 클러스터 목록
