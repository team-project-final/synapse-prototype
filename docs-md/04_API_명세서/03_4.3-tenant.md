# 4.3 Tenant 도메인

### GET /tenant/context

현재 테넌트 정보 조회

```json
{
  "data": {
    "tenantId": "uuid",
    "name": "My Workspace",
    "plan": "pro",
    "role": "owner",
    "quotas": {
      "notes": { "used": 45, "max": -1 },
      "aiGenerations": { "used": 120, "max": 500 }
    }
  }
}
```

### POST /tenant/switch

테넌트 전환 (복수 테넌트 멤버)

### GET /tenant/usage

사용량 현황 조회

### GET /tenant/members

멤버 목록 조회 (Team 플랜 이상)

### POST /tenant/members/invite

멤버 초대
