# 4.5 사용자 데이터 권리 (GDPR/CCPA)

### POST /me/data-export

데이터 내보내기 요청

```json
// Request
{ "format": "obsidian-zip", "scope": "all" }

// Response 202
{
  "data": {
    "jobId": "uuid",
    "status": "pending",
    "estimatedMinutes": 5
  }
}
```

### GET /me/data-export/{jobId}

내보내기 상태 조회

### DELETE /me/account

계정 삭제 요청 (30일 유예, GDPR Article 17)
