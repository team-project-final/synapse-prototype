## 1. 환경 매트릭스

| 항목 | Local | Dev | Staging | Prod |
|------|-------|-----|---------|------|
| **목적** | 개발/디버깅 | 통합 테스트 | QA/릴리즈 후보 | 실서비스 |

## 4. 초기화 SQL (PostgreSQL)

```sql
-- infra/docker/postgres/init.sql

-- pgvector 확장 활성화
CREATE EXTENSION IF NOT EXISTS vector;
CREATE EXTENSION IF NOT EXISTS "uuid-ossp";

-- RLS 활성화를 위한 기본 설정
ALTER DATABASE synapse SET app.current_tenant_id = '';

-- 테넌트 격리를 위한 공통 함수
CREATE OR REPLACE FUNCTION current_tenant_id()
RETURNS UUID AS $$
  SELECT NULLIF(current_setting('app.current_tenant_id', true), '')::UUID;
$$ LANGUAGE SQL STABLE;
```

## 7. 데모 데이터 시드

팀 프로젝트 평가 데모 시 빈 DB가 아닌 실감나는 데이터로 시연하기 위한 시드 스크립트입니다. `docker compose up` 후 실행합니다.

### 실행 방법

```bash
# Local 환경 시드 실행
psql -h localhost -U synapse_user -d synapse -f scripts/seed-demo.sql
```
