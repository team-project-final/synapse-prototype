> **기술 스택**: Spring Boot 4, Flutter 3.x, FastAPI, PostgreSQL 16, Redis, Elasticsearch, Kafka, K8s

## 1. 시작하기

### 1.1 회원가입

1. [synapse.app](https://synapse.app)에 접속합니다.
2. **"무료로 시작하기"** 버튼을 클릭합니다.
3. 이메일 주소와 비밀번호를 입력합니다.
   - 비밀번호는 8자 이상, 영문+숫자+특수문자 조합
   - 또는 Google/Apple 계정으로 간편 가입 가능
4. 이메일로 전송된 인증 링크를 클릭하면 가입이 완료됩니다.

### 1.2 첫 로그인

가입 후 첫 로그인 시 간단한 설정 과정이 진행됩니다:

1. **관심 분야 선택**: 학습 목적에 맞는 태그를 선택하세요 (예: 프로그래밍, 영어, 의학)
2. **복습 시간 설정**: 하루 중 복습 알림을 받고 싶은 시간대를 설정합니다
3. **튜토리얼 시작**: 3분 가이드 투어로 핵심 기능을 빠르게 익힐 수 있습니다

### 1.3 튜토리얼

첫 로그인 시 자동으로 시작되는 대화형 튜토리얼을 통해 다음을 배울 수 있습니다:

- 노트 작성 방법
- 위키링크로 노트 연결하기
- AI로 학습 카드 자동 생성하기
- 복습 세션 시작하기

> 💡 **팁**: 튜토리얼을 건너뛰었다면, 설정 > 도움말 > 튜토리얼 다시보기에서 언제든 다시 볼 수 있습니다.

## 2. 학습자 기능

### 2.1 노트 작성

Synapse의 노트는 마크다운 형식으로 작성합니다. 복잡한 문법을 몰라도 괜찮습니다.

**기본 작성법**:
- 새 노트: 왼쪽 사이드바 **"+ 새 노트"** 클릭 또는 `Ctrl+N`
- 제목 입력 후 본문을 자유롭게 작성
- 자동 저장됩니다 (별도 저장 불필요)

**서식 도구**:
| 기능 | 단축키 | 설명 |
|------|--------|------|
| 굵게 | `Ctrl+B` | **이렇게** 강조됩니다 |
| 기울임 | `Ctrl+I` | *이렇게* 표시됩니다 |
| 제목 | `Ctrl+1~6` | 제목 크기 조절 |
| 목록 | `Ctrl+L` | 글머리 기호 목록 |
| 코드 | `` Ctrl+` `` | `코드` 형태로 표시 |

### 2.2 위키링크로 노트 연결하기

노트 간의 연결이 Synapse의 핵심입니다. 위키링크를 사용해 지식을 연결하세요.

**사용법**: 본문에서 `[[`를 입력하면 기존 노트 목록이 나타납니다.

```
오늘 [[머신러닝]] 기초를 공부했다.
[[경사하강법]]이 [[선형회귀]]에서 사용된다는 것을 알게 되었다.
```

- `[[노트 제목]]`을 입력하면 해당 노트로 연결됩니다
- 존재하지 않는 제목을 입력하면 새 노트가 자동 생성됩니다
- 연결된 노트는 그래프 뷰에서 시각적으로 확인할 수 있습니다

### 2.3 AI 카드 생성

노트를 작성한 후, AI가 자동으로 학습 카드를 만들어줍니다.

**사용 방법**:
1. 노트를 열고 상단의 **"AI 카드 생성"** 버튼 클릭
2. AI가 노트 내용을 분석하여 질문-답변 쌍을 생성합니다
3. 생성된 카드를 검토하고 수정/삭제할 수 있습니다
4. **"카드 저장"**을 눌러 복습 덱에 추가합니다

**카드 유형**:
- **기본형**: 앞면(질문) → 뒷면(답변)
- **빈칸 채우기**: 핵심 단어가 빈칸으로 표시
- **다지선다**: 4개 선택지 중 정답 고르기

> 💡 **팁**: 노트가 구체적이고 체계적일수록 AI가 더 좋은 카드를 만들어줍니다.

### 2.4 복습 세션

Synapse는 과학적인 간격 반복(Spaced Repetition) 방식으로 복습 일정을 관리합니다.

**복습 시작하기**:
1. 메인 화면의 **"오늘의 복습"** 카드를 클릭합니다
2. 카드 앞면을 보고 답을 떠올립니다
3. **"정답 보기"**를 눌러 답을 확인합니다
4. 난이도를 평가합니다:
   - 😰 **어려움**: 내일 다시 복습
   - 🤔 **보통**: 3~7일 후 복습
   - 😊 **쉬움**: 2~4주 후 복습

**복습 통계**:
- 홈 화면에서 오늘의 복습 카드 수, 연속 학습일, 전체 진행률을 확인할 수 있습니다

### 2.5 그래프 뷰

그래프 뷰는 노트 간의 연결을 시각적으로 보여줍니다.

**사용법**:
1. 왼쪽 사이드바에서 **"그래프"** 아이콘 클릭
2. 노트가 점(노드)으로, 연결이 선(엣지)으로 표시됩니다
3. 노드를 클릭하면 해당 노트로 이동합니다
4. 마우스 휠로 확대/축소, 드래그로 이동

**필터 기능**:
- 태그별 필터링
- 연결 수 기준 필터링
- 특정 노트 중심으로 보기 (로컬 그래프)

### 2.6 검색

Synapse는 강력한 한국어 검색을 지원합니다.

**검색 방법**:
- 상단 검색바 클릭 또는 `Ctrl+K`
- 검색어 입력 시 실시간으로 결과가 표시됩니다

**검색 기능**:
- **전문 검색**: 노트 본문 내 모든 텍스트 검색
- **태그 검색**: `#태그명`으로 태그 필터링
- **AI 시맨틱 검색**: 정확한 단어가 아니어도 의미가 비슷한 노트 검색 (Pro 플랜)

### 2.7 데이터 내보내기

내 데이터는 언제든 내보낼 수 있습니다.

**내보내기 형식**:
- **Markdown**: 모든 노트를 .md 파일로 다운로드
- **JSON**: 카드 및 복습 데이터 포함
- **Anki 호환**: .apkg 파일로 Anki에서 사용 가능

**방법**: 설정 > 데이터 관리 > 내보내기

## 3. 커뮤니티 기능

### 3.1 스터디 그룹

- **그룹 만들기**: 사이드바 → 커뮤니티 → + 그룹 만들기
- **그룹 가입하기**: 커뮤니티 → 그룹 탐색 → 가입 버튼
- **멤버 초대하기**: 그룹 상세 → 멤버 → 초대
- **그룹에서 나가기**: 그룹 상세 → 설정 → 탈퇴

### 3.2 덱/노트 공유

- **덱 공유하기**: 덱 상세 → 공유 버튼 → 공유 대상 선택 (공개/그룹/링크)
- **공유 덱 찾기**: 커뮤니티 → 공유 덱 탐색 → 검색/필터
- **공유 덱 복사하기**: 공유 덱 상세 → 복사 버튼
- **덱 평가하기**: 공유 덱 상세 → 별점 + 댓글

### 3.3 게이미피케이션

- **내 프로필 확인**: 사이드바 → 프로필 → 레벨/XP/배지
- **배지 갤러리**: 프로필 → 배지 탭 → 전체 배지 확인
- **리더보드 보기**: 사이드바 → 리더보드 → 전체/그룹/주간/월간
- **XP 얻는 방법**: 복습 완료(+5), 노트 작성(+10), 카드 생성(+5), 덱 공유(+15), 연속 학습 보너스(+10 x 일수, 최대 100)

### 3.4 알림 설정

- **알림 확인**: 상단 바 → 🔔 아이콘 → 알림 센터
- **알림 설정 변경**: 설정 → 알림 → 카테고리별 on/off
- **방해금지 설정**: 설정 → 알림 → 방해금지 시간대

### 3.5 콘텐츠 신고

- **신고 방법**: 콘텐츠 → ··· 메뉴 → 신고하기 → 사유 선택

## 4. 설정

### 4.1 프로필 설정

- **표시 이름**: 다른 사용자에게 보이는 이름
- **프로필 사진**: 이미지 업로드 또는 기본 아바타
- **타임존**: 복습 알림 시간 기준
- **언어**: 인터페이스 언어 (한국어/English)

### 4.2 알림 설정

| 알림 유형 | 기본값 | 설명 |
|-----------|--------|------|
| 복습 리마인더 | 켜짐 | 매일 설정된 시간에 알림 |
| 주간 리포트 | 켜짐 | 매주 월요일 학습 통계 |
| 업데이트 소식 | 켜짐 | 새 기능 안내 |
| 이메일 알림 | 켜짐 | 중요 알림 이메일 발송 |

### 4.3 테마 설정

- **라이트 모드**: 밝은 배경 (기본)
- **다크 모드**: 어두운 배경 (눈의 피로 감소)
- **시스템 설정 따르기**: 기기 설정에 맞춰 자동 전환

### 4.4 SRS(복습) 설정

고급 사용자를 위한 복습 알고리즘 설정:

- **하루 최대 복습 카드**: 기본 50장 (조절 가능)
- **하루 최대 새 카드**: 기본 20장
- **쉬움 간격 배수**: 기본 2.5배
- **최대 간격**: 기본 365일

> 💡 **팁**: 기본 설정으로 충분히 효과적입니다. 특별한 이유가 없다면 변경하지 않는 것을 권장합니다.

### 5.2 업그레이드

1. 설정 > 구독 관리 > **"업그레이드"** 클릭
2. 원하는 플랜 선택
3. 결제 정보 입력 (신용카드/체크카드)

## 6. 자주 묻는 질문 (FAQ)

### 일반

**Q: 모바일에서도 사용할 수 있나요?**
A: 네, iOS와 Android 앱을 제공합니다. 앱스토어에서 "Synapse"를 검색하세요. 웹과 모바일 간 실시간 동기화됩니다.

**Q: 오프라인에서도 사용 가능한가요?**
A: 모바일 앱은 오프라인 모드를 지원합니다. 오프라인에서 작성한 내용은 인터넷 연결 시 자동 동기화됩니다.

**Q: 내 데이터는 안전한가요?**
A: 모든 데이터는 암호화되어 저장되며, 한국(서울) 리전의 AWS 서버에 보관됩니다. 정기 백업을 수행합니다.

### 노트/카드

**Q: 기존에 사용하던 노트를 가져올 수 있나요?**
A: Markdown(.md) 파일과 Anki(.apkg) 파일 가져오기를 지원합니다. 설정 > 데이터 관리 > 가져오기에서 이용하세요.

**Q: AI가 만든 카드가 마음에 들지 않으면?**
A: 카드 생성 후 자유롭게 수정하거나 삭제할 수 있습니다. AI는 초안을 만들어주는 도우미 역할입니다.

**Q: 위키링크를 잘못 만들었는데 어떻게 삭제하나요?**
A: 노트 본문에서 `[[링크]]` 텍스트를 지우면 연결이 해제됩니다.

### 결제

**Q: 무료 체험 기간이 있나요?**
A: Free 플랜을 제한 없이 사용할 수 있습니다. Pro 플랜은 7일 무료 체험 후 결제가 시작됩니다.

**Q: 환불이 가능한가요?**
A: 결제일로부터 7일 이내 요청 시 전액 환불됩니다. 설정 > 구독 관리 > 환불 요청에서 진행하세요.

## 7. 문제 해결

### 로그인이 안 됩니다

1. 이메일 주소가 정확한지 확인하세요
2. 비밀번호를 잊었다면 "비밀번호 찾기"를 이용하세요
3. 소셜 로그인(Google/Apple)으로 가입했다면 해당 방식으로 로그인하세요
4. 그래도 안 되면 support@synapse.app으로 문의하세요

### 동기화가 안 됩니다

1. 인터넷 연결을 확인하세요
2. 앱을 완전히 종료 후 다시 실행하세요
3. 설정 > 동기화 > "수동 동기화" 버튼을 눌러보세요
4. 문제가 지속되면 앱을 삭제 후 재설치하세요 (데이터는 서버에 보존됩니다)

### AI 카드 생성이 안 됩니다

1. 노트 내용이 너무 짧지 않은지 확인하세요 (최소 100자 권장)
2. Free 플랜 월 10회 한도를 확인하세요
3. 인터넷 연결이 불안정하면 AI 기능이 제한될 수 있습니다

### 검색 결과가 이상합니다

1. 검색어의 맞춤법을 확인하세요
2. 더 짧은 키워드로 검색해보세요
3. 태그 검색 시 `#` 기호를 포함하세요

### 앱이 느립니다

1. 앱을 최신 버전으로 업데이트하세요
2. 기기 저장 공간이 충분한지 확인하세요
3. 다른 앱을 종료하여 메모리를 확보하세요
4. Wi-Fi 환경에서 이용해보세요
